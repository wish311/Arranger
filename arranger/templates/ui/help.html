What Arranger does

Arranger audits Radarr movies and Sonarr series, matches metadata rules, and queues API-driven root-folder updates.

Dry-run

Dry-run means every destructive update is blocked. Arranger records what would happen so you can review safely.

Why API-only moves matter

Radarr/Sonarr own their databases and import paths. Arranger updates those apps via API and lets them move files so future imports remain correct.

Sonarr future episodes

After Sonarr accepts a series path/root update, future episodes import to that new path. The safety gate avoids active queues, missing monitored episodes, and recent imports.

Webhooks

Configure Radarr to call /webhook/radarr and Sonarr to call /webhook/sonarr. Webhooks never bypass dry-run or safety.

Troubleshooting

  • Target root not found: add it in Radarr/Sonarr root folders.
  • Permission issues: make sure the Arr app container can write target roots.
  • Queue active: wait for downloads/imports to finish.
  • Schema mismatch: verify your Arr version supports the expected API update shape.
